# STAGING — SCRATCH ZONE

> Draft/WIP output the AI is not yet confident about. Never a permanent home for anything.

---

## Rule
- If a task produces an intermediate artifact that isn't the final answer (a draft schema, a plan
  being validated, an exploratory spike), it goes here — not into `memory/` or `decisions/`.
- Nothing in this folder is authoritative. `governance/rules.md` authority hierarchy does not
  apply here — treat everything here as disposable.
- Clear a file out of staging once its content is either promoted (moved into the real `_brain/`
  file it belongs in) or discarded. Don't let this folder accumulate.
